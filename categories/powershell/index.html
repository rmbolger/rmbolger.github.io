Posh-ACME 3.5.0

New plugin and misc fixes

June 21, 2019 Ryan Bolger

Just shipped a new Posh-ACME release, version 3.5.0. This one has an important fix due to a recent change in Let’s Encrypt’s ACME implementation to be more compliant with the recently finalized RFC 8555 spec. The Let’s Encrypt change is only on the staging server at the moment, but it will likely move to production soon and it breaks account creation on all previous versions of Posh-ACME.

Posh-IBWAPI 3.0.0

File upload/download wrappers, named configs, and auto session management

April 20, 2019 Ryan Bolger

Just shipped a new Posh-IBWAPI release, version 3.0.0. I realize it has only been two days since 2.0.0, but I totally goofed and shipped 2.0.0 with some breaking changes that I ended up reverting because I’m an idiot. The biggest feature of the release is the new file upload/download wrappers, Send-IBFile and Recieve-IBFile. They allow you to more easily do things like upload a certificate or download a grid backup and fill a functionality gap that has been bugging me since 1.0.
